扩展运算符在函数中的使用

实际上就是将剩余参数处理为数组

与 arguments 的区别

arguments 为类数组，自动存在，包含所有的参数

什么是类数组

类数组的类型是对象，数组的类型是数组对象

类数组存在长度，可以通过索引访问，但是没有数组对象特有的功能，比如 pop()(删除最后一项，返回删除的值，原数组对象发生变化)

Array.isArray()为 false

类数组转换为数组的方法

Array.from(arguments)
[...arguments]
Array.prototype.slice.call(arguments)

为什么 Array.prototype.slice.call(arguments)可以生效

slice 是提取数组的方法
参数为提取数组的索引，返回一个新数组
slice 只需要 length 和能够通过索引访问就可以生效，使用 call 改变 this 的指向到 arguments 中
