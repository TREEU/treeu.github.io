HelloTriangele

navigator?.gpu

webgpu API 的入口点

adapter

可以理解为显卡硬件本身

featureLevel

按照官方文档，默认为 ‘core’
这是一个保留属性，从官方文档上看，不管设置为‘core’还是’compatibility’都没有任何不同

device

显卡驱动程序接口，获取得到 device 之后，就可以进行真正的 GPU 操作
比如创建 buffer，创建管线之类的

显卡驱动程序接口

显卡驱动接口是操作系统或应用程序与显卡硬件之间的软件层，它允许软件（如应用程序、游戏或 WebGPU 这样的图形 API）与显卡进行通信，而无需直接处理硬件的复杂性。

AI 总结

显卡驱动接口就是：

🎯 翻译官：将高级 API 调用转换为硬件指令

🛡️ 安全卫士：防止应用程序直接操作硬件

📊 资源经理：管理 GPU 内存和计算资源

🔄 抽象层：让代码能在不同显卡上运行
